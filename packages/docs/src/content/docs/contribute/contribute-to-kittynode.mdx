Kittynode is open to everyone, and happily welcomes any contributions! ヽ(・∀・)ﾉ

## Prerequisites

<Steps>

1. Install [Rust](https://www.rust-lang.org/):

    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. Install [just](https://github.com/casey/just):

    ```bash
    cargo install just
    ```

3. Install [bun](https://bun.sh/):

    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```

</Steps>

## Development

<Steps>

1. Clone the repo:

    ```bash
    git clone git@github.com:kittynode/kittynode.git
    ```

2. Install dependencies:

    ```bash
    bun install
    ```

3. Run repo commands with just:

    ```bash
    # Build the crates
    just build

    # Run the tests
    just test

    # Run the app
    just tauri

    # Run the docs
    just docs
    ```

4. See the full list of commands in the [justfile](https://github.com/kittynode/kittynode/blob/main/justfile).

</Steps>
